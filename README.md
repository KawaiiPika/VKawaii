# VKawaii

3D VTubing software built in Rust using Blue Engine.

## Features

- Custom `.vkw` asset format
- Advanced Physics Spring Bones System
- Blue Engine / WGPU backend
- Egui Overlay Editor

## Building

Make sure you have Rust installed. 

```sh
cargo build --release --bin vkawaii
```

To run:
```sh
cargo run --release --bin vkawaii
```
